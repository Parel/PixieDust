Class Detail
Engine(I)
The Engine controls the game world and manages game state. Once you set it up and let it run it pretty much takes care of itself. You can use the engine to add or remove objects from the game world. There are several modules that can include to add additional capabilities to the engine. The engine fires events that you may bind listeners to. Event listeners may be bound with engine.bind(eventName, callback) beforeAdd(entityData) Observer or modify the entity data before it is added to the engine. afterAdd(gameObject) Observe or configure a gameObject that has been added to the engine. draw(canvas) Called after the engine draws on the canvas, you wish to draw additional things to the canvas. update Called after the engine updates all the game objects.
